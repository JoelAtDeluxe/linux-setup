# linux-setup

Ansible scripts to help me set up my personal linux set up, targeting:

* Ubuntu 18.04
* Fedora 28

## Prereqs
As these are ansible scripts, a necessary component is ansible. 

* Install:
    * ansible
* Run as:
    * root

### Notes for Fedora

* Install:
    * python2-dnf

### Pre-reqs for Ubuntu

* Install:
    * python-apt

## Running

Simply run `ansible-playbook master-playbook.yml` to start the process
